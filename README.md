## 🇬🇧 English

### About The Project
A professional local service website developed specifically for carpentry and custom furniture manufacturing in Riyadh, Saudi Arabia. The project is heavily optimized for Local SEO, features a modern UI/UX, and provides seamless WhatsApp integration for instant customer communication.

### Key Features
- **100% Responsive Design:** Flawless experience on desktops, tablets, and mobile devices.
- **Local SEO Optimized:** Includes Schema Markup (LocalBusiness, FAQ, Service) and Meta tags.
- **PWA Ready:** Manifest and icons configured for a native app-like experience.
- **WhatsApp Integration:** Direct lead generation with pre-filled forms sending data directly to WhatsApp.
- **Fast Performance:** Clean code architecture ensuring rapid loading speeds.

### Built With
- HTML5
- CSS3
- Vanilla JavaScript
- JSON-LD (Structured Data)
